# 치킨어때 🍗
나만 먹기 아까운 치킨들을 남들에게 추천하고싶은 적 있지 않나요? 혹은 어떤 치킨을 먹을지 메뉴판을 보며 고민한적 있지 않나요?
  
  __치킨어때__ 는 나만먹기 아까운 맛있는 치킨들을 공유하고 리뷰를 남기면서 치킨을 고를 때 참고할 수 있게 서로의 의견을             공유하는 플랫폼 입니다.

---------------------------------------

#### 목차
* 사용 기술
* 팀원 목록
